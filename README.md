## `rust-bin-reloader`

**Usage:** `rust-bin-reloader [OPTIONS] <PATH> [-- <BINARY_ARGS>...]`

###### **Arguments:**

* `<PATH>` — Path to binary file
* `<BINARY_ARGS>` — Additional arguments to pass to the binary

###### **Options:**

* `-d <RESTART_DELAY>` — Delay when restarting the binary after an unexpected exit

  Default value: `3s`
